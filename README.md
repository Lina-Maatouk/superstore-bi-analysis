# Superstore Sales Analysis

This repository contains a business intelligence analysis of a US retail superstore dataset using Python.

## Project Overview

- Dataset: `train.csv`
- Notebook: `sales_project.ipynb`
- Tools: Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly, scikit-learn

## Contents

- `sales_project.ipynb` — full analysis notebook with data cleaning, exploratory data analysis, time series visualization, product performance, regional analysis, and forecasting.
- `train.csv` — input dataset.
- Plot images generated from the analysis:
  - `monthly_trend.png`
  - `monthly_trend_with_line.png`
  - `category_sales.png`
  - `region_sales.png`
  - `map.html`

## Key Insights

- Monthly sales show strong seasonality with Q4 revenue spikes.
- Certain product categories and regions drive the bulk of sales.
- A linear regression trend line is used to model monthly sales trajectory.

## Usage

1. Open `sales_project.ipynb` in Jupyter Notebook or JupyterLab.
2. Run the notebook cells sequentially.
3. View the generated charts and analysis results.
